Desktop AI Pet

다운로드 / Downloads

Desktop AI Pet 실행 파일을 받을 수 있는 페이지입니다. Windows와 macOS용 빌드를 아래에서 선택해 다운로드하세요.

최신 버전 정보를 불러오는 중…

macOS에서 “손상됨”이 뜰 때

앱이 깨진 것이 아닙니다. 다운로드 파일에 붙는 격리(quarantine) 때문입니다. 우클릭 열기 / ZIP 안 스크립트 더블클릭만으로는 안 되는 경우가 많습니다. 앱을 다른 폴더로 옮길 필요는 없습니다. 실제 .app 경로만 맞추면 됩니다.

ZIP을 풀면 보통:

~/Downloads/DesktopAIPet.app
# 또는
~/Downloads/DesktopAIPet-macos-arm64/DesktopAIPet.app

터미널(응용 프로그램 → 유틸리티 → 터미널)에 경로에 맞게 붙여넣기:

xattr -cr ~/Downloads/DesktopAIPet.app && open ~/Downloads/DesktopAIPet.app

# 폴더 안에 있으면:
xattr -cr ~/Downloads/DesktopAIPet-macos-arm64/DesktopAIPet.app && open ~/Downloads/DesktopAIPet-macos-arm64/DesktopAIPet.app

경로가 헷갈리면 터미널에 xattr -cr (끝에 공백)을 친 뒤, Finder의 DesktopAIPet.app을 터미널로 끌어다 넣고 Enter 하세요.
